# ada-rn-wrapper

React Native Wrapper for ada js lib

## Installation

```sh
npm install ada-rn-wrapper
```

## Usage

```js
import AdaRnWrapper from "ada-rn-wrapper";

// ...

const result = await AdaRnWrapper.multiply(3, 7);
```
